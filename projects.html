HTML

HTML (HyperText Markup Language) is the foundational language used to create and structure webpages on the internet. It uses a system of tags to define elements such as headings, paragraphs, links, images, and more. A typical HTML document is divided into two main sections: the head, which contains metadata and links to external resources like CSS files, and the body, which contains all the visible content shown on the webpage.

CSS

CSS (Cascading Style Sheets) is used to style and design HTML webpages. It controls colors, fonts, spacing, layouts, and animations. CSS can be written inline, internal, or in an external stylesheet. It works with HTML to make websites look attractive and user-friendly.

BOOTSTRAP

Bootstrap is a free and open-source CSS framework used to build responsive and mobile-first websites. It provides ready-made components like buttons, navbars, cards, modals, etc. Bootstrap uses a grid system (12-column layout) for easy page layout design. It saves time by using predefined classes instead of writing custom CSS from scratch.

JAVA SCCRIPT

JavaScript (JS) is a programming language used to add interactivity to websites. It can change HTML content, handle user events (like clicks), and validate forms. JS works with HTML and CSS to create dynamic and responsive web pages. It runs directly in the browser and is essential for modern web development.
